# University Campus Guide - Full Stack Project

A full-stack MERN campus guide web application with user authentication, campus location data, and map-based navigation.

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- bcrypt Password Hashing
- React Leaflet / Leaflet Map View

## Features

- User registration and login
- Secure password hashing
- JWT-based authentication
- Campus location listing
- Interactive map view
- Backend MVC architecture
- Seed file for inserting initial campus data
